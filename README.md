## vert.X web application for keep younger

#### 项目实现
* 1.文件读取
* 2.自定义拦截器
* 3.子路由
* 4.错误处理器
* 5.权限控制
* 6.简易文件服务器

#### 启动
启动参数为一个密钥，详情参见代码
* 开发时启动：
    > 如果时IDEA开发，请配置启动时参数: \
     `Edit Configurations` -> `Program Arguments` -> 添加启动参数
* jar包启动
    > 项目打包成jar后：\
    java -jar keep-younger-web-api-1.1.0-prod.jar xxx \
    xxx 为输入的启动参数
